# Ergocode AI Agent Framework

Фреймворк задаёт роли, скиллы и минимальные правила работы ИИ-агентов,
чтобы они проектировали и писали код по Эргономичному подходу.

## Цель

- сократить путь от задачи до кода;
- вынести переиспользуемые правила из проектов;
- сделать поведение агента более предсказуемым.

## Состав

- `.agents` — файлы управляющие работой агента разработки фреймворка;
- `bootstrap` — файлы установки фреймворка в целевой репозиторий, используются один раз; 
- `src/` — файлы для установки в целевой репозиторий;

## Установка

1. Добавить фреймворк в проект.
   Скопировать, сделать symlink на него или добавить git submodule.
   По умолчанию фреймворк должен находиться в `.agents/ergo`.
2. Запустить скилл установки промптом `Install the framework in this repository using [SKILL.md](.agents/ergo/bootstrap/skills/installing-framework/SKILL.md).`.
3. Если файла `AGENTS.md` ранее не существовало - дополнить его проектными инструкциями, например с помощью специализированного скилла вашего агента.
